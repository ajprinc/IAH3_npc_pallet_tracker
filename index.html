NPC Work Types

Quick reference

1) Decant Pallet

  • Pallet received - "PALLET"/"PALLETLP" in FC Research
  • Multi ASIN (mixed items)
  • Has CSX labels - normal decant
  • No CSX labels - Carton Preditor first, then decant

2) Each Receive Pallet

  • NOT pallet received (must be each received)
  • No inventory attached when you scan paX
  • Sometimes requires prep

3) Pre-sort Pallet

  • Pallet received (like decant)
  • Single ASIN (not mixed)
  • Non-PID conveyable

4) Dock Door / Outbound

  • Single destination - goes straight to dock door

FC Research Lookup

  • qi-fcresearch-na.corp.amazon.com
  • Search paX - Receive History - Process Path
  • "PALLET"/"PALLETLP" = pallet received
  • No history = Each Receive
